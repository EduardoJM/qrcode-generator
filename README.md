## Funcionalidades

- Gerar QR Code.
- Ajuste de cores.
- Copiar para a área de transferência.
- Baixar imagem.
- Moldura do QRCode.

## Rodando localmente

Clone o repositório:

```bash
git clone https://github.com/EduardoJM/qrcode-generator.git
```

Instale as dependências

```bash
cd qrcode-generator
npm install
```

Rode a versão de desenvolvimento

```bash
npm run dev
```
